
# Doggies - Individual Project

SPA (Single Page Application) about dog breeds, for the individual project of the SoyHenry Bootcamp.
The application allows you to see different dog breeds along with relevant information about them by consuming an external API, and from it, among other things, you can: Search for dog breeds, Filter/Sort them, Add new dog breeds.
To achieve the final product, I used:
- Node.js together with Express.js to develop the application routes, and consume data from, both the external API and the local database.
- PostgreSQL and Sequelize to save some data from the external API, and the new breeds of dogs created by the user.
- React in conjunction with Redux for the development of the Frontend, and the management of local and global states.
- HTML and CSS Modules for user interface design.
